# Code of BA-SGCL
Code of "Balance Augmented-Signed Graph Contrastive Learning"

In the folder, we provide 4 clean signed graph datasets mentioned in our paper. To evaluate BA-SGCL's performance, it is recommended to apply Balance-attack and FlipAttack on the clean datasets beforehand. Change the input data both in "train.py" and "/utils/edge_data_sign.py".

# Run
python train.py
